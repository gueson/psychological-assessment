# 心理测评工具全解析

## 项目简介

心理测评工具全解析是一个专业的心理评估指南平台，专注于探索人格测试、能力测试和职业兴趣测试的理论基础与实践应用。本项目旨在为用户提供全面、专业的心理测评工具信息，帮助了解不同类型的心理测评及其应用场景。

## 核心功能

- **人格测试解析**：详细介绍MBTI、大五人格、艾森克人格问卷等主流人格测评工具
- **能力测试详解**：涵盖智力测试、特殊技能评估、管理能力测评等32种能力测试类型
- **职业兴趣测试指南**：解析霍兰德职业兴趣测试、斯特朗职业兴趣量表等职业规划工具
- **专业评估建议**：提供科学的心理测评使用建议和最佳实践
- **响应式设计**：适配各种设备尺寸，提供良好的用户体验

## 技术栈

- **前端框架**：Next.js 14
- **样式方案**：Tailwind CSS
- **语言**：TypeScript
- **部署**：支持Vercel、Netlify等静态网站托管服务

## 快速开始

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000 查看项目运行效果。

```bash
npm run build
npm run start
```
